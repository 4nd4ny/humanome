# Lire une cartographie partagée

Manuel de prise en main pour l'**employeur / recruteur**. Un candidat vous a
partagé sa cartographie : comment l'ouvrir, la lire correctement, comprendre la
garantie d'un cartographe — et les limites d'un tel document.

Les chapitres se lisent dans l'ordre depuis le [hub des guides](#/guides).
